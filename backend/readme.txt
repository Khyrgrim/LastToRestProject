Différent packages son nécéssaire :
    Pour le frontend (à entré dans la console après avoir trouver cd ....../frontend): 

        npm install

    Pour le back end (à entré dans la console après avoir trouver cd ....../backend):

        npm install -g nodemon
        npm install express
        npm install mongoose
        npm install mongoose-unique-validator
        npm install bcrypt
        npm install jsonwebtoken



Pour lancer le frontend (à entré dans la console après avoir trouver cd ....../frontend)):

    npm run start

Pour lancer le backend (à entré dans la console après avoir trouver cd ....../backend):

    node server
